Round Sequence

Reference
  1. Select one phase secretly.
  2. Reveal selected phases.
  3. Resolve selected phases in order.
  4. Manage Empire: spend Credits to recharge.

No Roll Dice step. No Assign Dice step. No Cup or Citizenry.

Spend Pips

Reference
  • Blue: Explore worker.
  • Brown: Develop worker.
  • Red: Settle worker.
  • Green: Produce worker.
  • Purple: Ship worker.
  • White: any selected phase.

Gain Dice

Reference

When a tile says to gain a die, pip up that color.

  1. Increase max by 1, max 6.
  2. Increase current by 1, not above max.
  3. Excess beyond 6 is lost.

Green 0/2 + Genes die = Green 1/3.

Credits

Reference
  • Credits do not recruit dice.
  • Spend 1 Credit to recharge 1 current pip.
  • Never recharge above max.
  • If Credits are 0 after Manage Empire, set to 1.

Produce / Ship

Reference
  • Produce: place a Good marker, not a worker die.
  • The Good color is the pip color spent.
  • Ship: spend Purple or White to Trade or Consume.
  • White Goods count as matching.

Alien Yellow

Reference
  • Yellow starts 0/0.
  • Gain Alien Technology die: pip up Yellow.
  • Yellow pays only for Alien Develop, Settle, Produce, or Ship work.
  • Alternative: Yellow as extra Ship capacity.

Tile Powers

Reference
  • Ignore reassign powers.
  • Gain/recruit/place die: pip up that color.
  • Extra worker: gain 1 temporary pip for that phase.
  • Cost reduction: reduce pips required.

End Game

Reference
  • Normal end conditions.
  • Normal VP chips and tile VP.
  • For bonuses that count dice owned, count max pips.
  • Red max 4 = 4 Military dice owned.
